👥 Gestion des Rôles et Permissions

👑 Super Admin

Accès complet à toutes les fonctionnalités. Peut gérer les autres administrateurs et modifier les paramètres système.

Permissions :
  • Toutes les permissions
  • Gérer les administrateurs
  • Modifier les rôles
  • Accès aux paramètres système
⚙️ Admin

Peut gérer les utilisateurs, voir les analytics et gérer le support. Accès à la plupart des fonctionnalités.

Permissions :
  • Gérer les utilisateurs
  • Voir les analytics
  • Gérer le support
  • Voir les logs
💬 Support

Accès limité au support client et consultation des utilisateurs. Idéal pour l'équipe support.

Permissions :
  • Voir les utilisateurs
  • Gérer le support
  • Répondre aux tickets
👁️ Viewer

Accès en lecture seule aux analytics et logs. Aucune modification possible.

Permissions :
  • Voir les analytics
  • Voir les logs
  • Lecture seule

ℹ️ Comment modifier le rôle d'un utilisateur ?

Pour modifier le rôle d'un administrateur :

  • 1. Allez dans la page Utilisateurs
  • 2. Cliquez sur Voir pour l'utilisateur concerné
  • 3. Dans la modal, sélectionnez le nouveau rôle
  • 4. Cliquez sur Modifier le rôle

⚠️ Seuls les Super Admins peuvent modifier les rôles des autres administrateurs.
